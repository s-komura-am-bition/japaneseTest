日本語能力試験N4レベル 文法問題

1. このペンはとても（　　）やすいです。

  1. 1. かき
  2. 2. かく
  3. 3. かいて
  4. 4. かこう
答えと解説

答え： 1. かき

解説：動詞のます形から「ます」を取った形に「やすい」を付けて、「〜しやすい」という意味になります。「書きます」→「書き」＋やすい

2. 私は昨日、友達に（　　）をもらいました。

  1. 1. はなを
  2. 2. はなが
  3. 3. はなで
  4. 4. はなも
答えと解説

答え： 1. はなを

解説：「もらう」という動詞は、もらう物事を表す名詞に助詞「を」を伴います。

3. 彼はまだ（　　）いません。

  1. 1. 来て
  2. 2. 来ます
  3. 3. 来ません
  4. 4. 来ました
答えと解説

答え： 1. 来て

解説：「まだ〜ていません」の形で、「まだ〜していない」という継続の否定を表します。

4. （　　）から、窓を閉めてください。

  1. 1. 寒い
  2. 2. 寒く
  3. 3. 寒くて
  4. 4. 寒かった
答えと解説

答え： 1. 寒い

解説：理由を表す「から」の前には、普通形が来ます。い形容詞の普通形はそのままです。

5. 私は日本語（　　）話せます。

  1. 1. が
  2. 2. を
  3. 3. は
  4. 4. も
答えと解説

答え： 1. が

解説：可能動詞「話せます」の前には、対象を表す助詞として「が」がよく使われます。

6. うるさいから、静かに（　　）ください。

  1. 1. して
  2. 2. する
  3. 3. しよう
  4. 4. しなさい
答えと解説

答え： 1. して

解説：「〜てください」は動詞のて形に接続して、依頼や指示を表します。「静かにする」のて形は「静かにして」です。

7. この本は（　　）そうです。

  1. 1. おもしろ
  2. 2. 面白く
  3. 3. 面白い
  4. 4. 面白
答えと解説

答え： 1. おもしろ

解説：様態の「〜そうです」（〜のように見える）は、い形容詞の「い」を取った形に接続します。「面白い」→「おもしろ」＋そうです。

8. 宿題は明日（　　）出してください。

  1. 1. までに
  2. 2. まで
  3. 3. ごろ
  4. 4. だけ
答えと解説

答え： 1. までに

解説：「までに」は「〜という期限までに」という、行為が完了する時点を示します。

9. 彼は日本語を（　　）のが上手です。

  1. 1. 話す
  2. 2. 話し
  3. 3. 話して
  4. 4. 話した
答えと解説

答え： 1. 話す

解説：動詞の辞書形に「の」を付けることで、「〜すること」という名詞句を作ることができます。「上手です」の前には名詞（句）が来ます。

10. 私はすしを（　　）ことがあります。

  1. 1. 食べた
  2. 2. 食べる
  3. 3. 食べて
  4. 4. 食べるの
答えと解説

答え： 1. 食べた

解説：「〜たことがあります」は、過去の経験を表す表現です。動詞のた形に接続します。
