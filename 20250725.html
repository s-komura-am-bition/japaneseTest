オフィス日本語 N4レベル 読解問題

問題 1

＜メモ＞ 田中さんへ あしたの 10時からの 会議は、午後 2時に なりました。 会議室は ３階のままです。 よろしくおねがいします。 山田

問題：会議は いつ ありますか。

  1. 1. あしたの 午前 10時
  2. 2. あしたの 午後 2時
  3. 3. あさっての 午前 10時
  4. 4. あさっての 午後 2時
答え： 2. あしたの 午後 2時

問題 2

＜メール＞ 件名：資料のお願い 佐藤さん 来週の 会議で 使う 資料を コピーして ください。 資料は 私の つくえの 上に あります。 コピーは 10部 おねがいします。 鈴木

問題：鈴木さんは 佐藤さんに 何を お願いしましたか。

  1. 1. 資料を 作ること
  2. 2. 資料を 10部 コピーすること
  3. 3. つくえの 上を そうじすること
  4. 4. 来週の 会議に 出ること
答え： 2. 資料を 10部 コピーすること

問題 3

＜会社のルール＞ 休憩室（きゅうけいしつ）を使うみなさんへ 休憩室で ごはんを 食べたあとは、かならず つくえを きれいに してください。 ゴミは ゴミ箱に すててください。 自分の ものは わすれないように しましょう。

問題：休憩室で ごはんを 食べたあと、何を しなければなりませんか。

  1. 1. すぐに 仕事に もどる
  2. 2. つくえを きれいにする
  3. 3. 自分の ものを おく
  4. 4. ゴミを もって 帰る
答え： 2. つくえを きれいにする

問題 4

（電話の会話） A：「はい、ABCフーズです。」 B：「もしもし、私、ミラーと もうします。いつも お世話に なっております。山本課長は いらっしゃいますか。」 A：「すみません、山本は 今、他の 電話に 出ております。」 B：「そうですか。では、また あとで かけます。」

問題：ミラーさんは なぜ 山本課長と 話せませんでしたか。

  1. 1. 山本課長が 休みだったから
  2. 2. 山本課長が 会議中だったから
  3. 3. 山本課長が 他の 電話に 出ていたから
  4. 4. ミラーさんが 電話を まちがえたから
答え： 3. 山本課長が 他の 電話に 出ていたから

問題 5

＜お知らせ＞ 社員のみなさんへ 来週の 月曜日、会社の パソコンが 新しくなります。 古い パソコンの 中の 大切な データは、金曜日までに USBメモリに 保存（ほぞん）して ください。

問題：金曜日までに 何を しなければなりませんか。

  1. 1. 新しい パソコンを 買う
  2. 2. データを USBメモリに 保存する
  3. 3. 古い パソコンを すてる
  4. 4. 月曜日に 会社を 休む
答え： 2. データを USBメモリに 保存する

問題 6

（オフィスの会話） A：「すみません、ちょっと いいですか。」 B：「はい、何ですか。」 A：「この エアコンの つけ方が わからないんですが…。」 B：「ああ、その リモコンの 赤い ボタンを 押せば つきますよ。」 A：「ありがとうございます。」

問題：Aさんは 何が わからなかったですか。

  1. 1. リモコンの ばしょ
  2. 2. エアコンの つけ方
  3. 3. 赤い ボタンの いみ
  4. 4. Bさんの なまえ
答え： 2. エアコンの つけ方

問題 7

＜メール＞ 件名：事務所の電気 みなさん 事務所を 最後に出る人は、かならず エアコンと 電気を けして ください。 窓のかぎも 確認（かくにん）してください。 ご協力（きょうりょく）ありがとうございます。

問題：最後に 事務所を 出る人が しなくてもいいことは どれですか。

  1. 1. エアコンを けすこと
  2. 2. 電気を けすこと
  3. 3. 窓のかぎを しめること
  4. 4. ドアの そうじを すること
答え： 4. ドアの そうじを すること

問題 8

＜メモ＞ 高橋さんへ 今日、宅急便（たっきゅうびん）が ２つ とどきます。 １つは 田中さんあて、もう１つは 会社あてのにもつです。 私がいないあいだに とどいたら、受け取っておいてください。 木村

問題：宅急便は だれあてに とどきますか。

  1. 1. 高橋さんと 木村さん
  2. 2. 高橋さんと 田中さん
  3. 3. 田中さんと 会社
  4. 4. 木村さんと 会社
答え： 3. 田中さんと 会社

問題 9

＜お知らせ＞ あしたの 朝９時から １０時まで、事務所の 窓の そうじを します。 そうじの人が 事務所の 中に 入りますから、びっくりしないで ください。 ご迷惑（めいわく）を おかけしますが、よろしく お願いします。

問題：お知らせの 内容（ないよう）は 何ですか。

  1. 1. 事務所の そうじの 時間
  2. 2. 新しい そうじの 人の しょうかい
  3. 3. 窓の しめ方
  4. 4. 事務所の ルール
答え： 1. 事務所の そうじの 時間

問題 10

（オフィスの会話） A：「加藤さん、コピー機の 紙が なくなりました。どこに ありますか。」 B：「ああ、紙は そこの たなの 中に ありますよ。」 A：「ありがとうございます。どの たなですか。」 B：「ドアの いちばん 近い たなです。」

問題：コピー機の 紙は どこに ありますか。

  1. 1. ドアの となり
  2. 2. コピー機の 中
  3. 3. ドアから いちばん とおい たな
  4. 4. ドアに いちばん 近い たな
答え： 4. ドアに いちばん 近い たな
